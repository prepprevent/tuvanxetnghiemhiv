Công cụ tổng hợp số liệu báo cáo TVXN

Import file dữ liệu KP eLog, chọn kỳ báo cáo, tự động tổng hợp chỉ số và xuất Excel.

1. Import data

Chọn một hoặc nhiều file Excel xuất từ KP eLog. Mỗi file cần có các sheet: DANH SACH KHACH HANG, KHACH HANG CHI DAN, BAN TINH - BAN CHICH.

Dữ liệu xử lý trên trình duyệt
Chưa chọn file.

2. Chọn kỳ báo cáo

Hãy import file data trước.

3. Kết quả tổng hợp

Danh sách CGTC_EPIC

Danh sách ca dương tính mới được kết nối điều trị thành công trong kỳ báo cáo.

Danh sách CGTC_LIFE

Danh sách ca thuộc chỉ số CGTC_LIFE trong kỳ báo cáo.

Phiên bản web chuyển đổi từ Power Query của file “Công cụ tổng hợp số liệu báo cáo TVXN_v2.0.xlsx”.
